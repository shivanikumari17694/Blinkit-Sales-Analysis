
📊 Blinkit Sales Dashboard

Built an interactive dashboard using Microsoft Power BI to analyse retail sales data and uncover key business insights across outlets, product categories, and customer behaviour.

🔍 Key Insights
Tier 3 outlets generated the highest revenue contribution
Fruits and snack categories were the top-performing segments
Sales peaked around 2018 and stabilised afterwards
Supermarket Type 1 outlets led in both sales and inventory

⚙️ Features
KPI tracking (Total Sales, Avg Sales, Ratings, Items)
Time-series analysis of sales trends
Category and outlet-level performance breakdown
Interactive filters for dynamic exploration

🛠️ Tools Used
Microsoft Power BI
Power Query (data cleaning & transformation)
DAX (calculated measures & KPIs)
